## 🎯 **操作步骤**

### 1. 复制组件文件

```
# 从 node_modules 复制到你的分包 components 目录
cp node_modules/uni-tree-view/src/components/uni-tree-view/uni-tree-view.vue src/packageC/components/
```

### 2. 在分包中创建组件目录

更好的组织方式是创建一个单独的目录：

```
src/packageC/
├── components/
│   ├── uni-tree-view/          # 创建专门目录
│   │   ├── uni-tree-view.vue   # 主组件
│   │   └── index.vue           # 可选：入口文件
│   ├── CategoryTree.vue
│   └── CategoryTreeNode.vue
└── pages/
    └── aioveuMallPms/
        └── CategoryTree/
            └── index.vue
```

### 3. 修改导入路径

在你的 `CategoryTree.vue`中：

```
// 从本地导入
import UniTreeView from '@/components/uni-tree-view/uni-tree-view.vue'

// 或者在 pages.json 中配置 easycom
```

### 4. 配置 Easycom（可选）

在 `pages.json`中配置分包级别的 easycom：

```
{
  "subPackages": [
    {
      "root": "packageC",
      "pages": [
        {
          "path": "pages/aioveuMallPms/CategoryTree/index",
          "style": { "navigationBarTitleText": "分类管理" }
        }
      ]
    }
  ],
  "easycom": {
    "autoscan": true,
    "custom": {
      // 分包内的组件路径
      "^uni-tree-view$": "src/packageC/components/uni-tree-view/uni-tree-view.vue"
    }
  }
}
```

## 💡 **更优方案：创建本地化组件**

与其复制整个 `uni-tree-view`，不如**提取它的核心逻辑，创建自己的简化版本**：

```
<!-- src/packageC/components/LocalTree.vue -->
<template>
  <view class="local-tree">
    <tree-node
      v-for="node in data"
      :key="node.id"
      :node="node"
      :level="0"
      :props="props"
      @node-click="$emit('node-click', $event)"
    />
  </view>
</template>

<script setup lang="ts">
import { defineProps, defineEmits } from 'vue'
import TreeNode from './TreeNode.vue'

defineProps<{
  data: any[]
  props?: {
    id?: string
    label?: string
    children?: string
  }
}>()

defineEmits(['node-click'])
</script>
```

```
<!-- src/packageC/components/TreeNode.vue -->
<template>
  <view>
    <view 
      class="tree-node"
      :style="{ paddingLeft: (level * 20) + 'rpx' }"
      @click="$emit('node-click', node)"
    >
      <text v-if="hasChildren" @click.stop="toggleExpand">
        {{ expanded ? '▼' : '▶' }}
      </text>
      <text>{{ node[labelKey] }}</text>
    </view>
    
    <view v-if="expanded && hasChildren">
      <TreeNode
        v-for="child in node[childrenKey]"
        :key="child[idKey]"
        :node="child"
        :level="level + 1"
        :props="props"
        @node-click="$emit('node-click', $event)"
      />
    </view>
  </view>
</template>

<script setup lang="ts">
import { ref, computed, defineProps, defineEmits } from 'vue'

const props = defineProps<{
  node: any
  level: number
  props?: {
    id?: string
    label?: string
    children?: string
  }
}>()

const emit = defineEmits(['node-click'])

// 计算键名
const idKey = computed(() => props.props?.id || 'id')
const labelKey = computed(() => props.props?.label || 'name')
const childrenKey = computed(() => props.props?.children || 'children')

// 展开状态
const expanded = ref(false)
const hasChildren = computed(() => 
  props.node[childrenKey.value] && props.node[childrenKey.value].length > 0
)

const toggleExpand = () => {
  if (hasChildren.value) {
    expanded.value = !expanded.value
  }
}
</script>

<style scoped>
.tree-node {
  padding: 20rpx;
  border-bottom: 1rpx solid #f0f0f0;
  background: white;
  display: flex;
  align-items: center;
}
</style>
```

## 📊 **优势对比**

| 方案                   | 优点                 | 缺点                       |
| ---------------------- | -------------------- | -------------------------- |
| **复制 uni-tree-view** | 功能完整，不用自己写 | 代码臃肿，可能有兼容问题   |
| **自己实现简单版本**   | 完全可控，代码精简   | 需要自己实现               |

## 🎯 **建议**

**自己实现简单版本**，因为：

1. 你的分类树需求简单（显示+展开/收起）
2. 完全可控，没有依赖问题
3. 代码精简，性能更好
4. 调试方便

**30 行代码就能搞定核心功能**，比折腾第三方组件快多了。
