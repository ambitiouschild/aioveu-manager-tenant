## ✅ 推荐方案

用最简单的三选项方案：

```javascript
// GoodsStock.vue
const handlePrev = () => {
  if (!checkStockChanges()) {
    emit("prev");
    return;
  }
  
  uni.showActionSheet({
    title: "有未保存的修改",
    itemList: ["💾 保存草稿并离开", "🗑️ 放弃修改并离开", "✏️ 继续编辑"],
    success: (res) => {
      switch (res.tapIndex) {
        case 0: // 保存草稿
          saveDraftToLocal();
          uni.showToast({ title: "草稿已保存", icon: "success" });
          setTimeout(() => emit("prev"), 1500);
          break;
          
        case 1: // 放弃修改
          emit("prev");
          break;
          
        case 2: // 继续编辑
          // 什么都不做
          break;
      }
    }
  });
};
```

这样用户就有清晰的选择：
1. 保存草稿：安全，可恢复
2. 放弃修改：快速离开
3. 继续编辑：误操作保护
