## 🎯 最终效果

| 图标 | 功能                   | 触发事件        |
| ---- | ---------------------- | --------------- |
| ✏️    | 编辑当前分类           | `@edit-click`   |
| ➕    | 在当前分类下新增子分类 | `@add-click`    |
| 🗑️    | 删除当前分类           | `@delete-click` |

**用户交互流程：**

1. 鼠标悬停到节点上 → 显示右侧操作按钮
2. 点击 ✏️ → 弹出编辑对话框
3. 点击 ➕ → 弹出新增下级对话框
4. 点击 🗑️ → 弹出删除确认
