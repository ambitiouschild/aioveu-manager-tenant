## ✅ 最小改动方案

### 1. 修改模板结构

找到 `flat-tree.vue`中的这部分代码：

```
<!-- 找到这行 -->
<view
  v-for="item in displayList"
  :key="item.id"
  class="tree-item"
  :style="{
    paddingLeft: (item.level * 40) + 'rpx',
    display: item._visible ? 'flex' : 'none'
  }"
>
```

改为：

```
<view
  v-for="item in displayList"
  :key="item.id"
  class="tree-item"
  :style="{
    paddingLeft: (item.level * 40) + 'rpx',
    display: item._visible ? 'flex' : 'none'
  }"
  @tap="handleItemClick(item)"  <!-- ✅ 添加这行：整行点击事件 -->
>
```

### 2. 添加点击处理方法

在 `flat-tree.vue`的 `<script setup>`中添加方法：

```
/**
 * 整行点击事件
 */
const handleItemClick = (item: FlatTreeNode) => {
  // 如果有子节点，切换展开状态
  if (item.children && item.children.length > 0) {
    toggleExpand(item)
  } else {
    // 没有子节点，触发正常的节点点击
    handleClick(item)
  }
}
```

## 🎯 工作原理

- 点击**有子节点的行** → 展开/收起
- 点击**没有子节点的行**（叶子节点）→ 触发原来的 `handleClick`事件
- 点击**操作按钮** → 因为有 `@tap.stop`，所以不会触发整行点击
