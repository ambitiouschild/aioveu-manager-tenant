### ❌ 1️⃣ APP‑PLUS 下载代码还有一处隐患（必须修）

```ts

```

在 Android 上偶尔会失败。

✅ 终局安全写法（强烈建议）

```ts

reader.onload = function () {
  writer.write(reader.result as ArrayBuffer);

reader.readAsArrayBuffer(blob);
```
