根据您提供的代码，我发现了问题所在。您圈出的红色边框是 **`uni-grid`组件自身的默认边框**，在您的代码中并没有通过属性或样式将其关闭。

## 问题分析

在您的代码中：

```
<uni-grid :column="4" :highlight="false" :square="false" class="grid-container">
```

这里缺少了关键的属性 `:show-border="false"`来关闭网格的边框。

## 解决方案

### 方案1：在 `<uni-grid>`上添加属性（推荐）

在您的模板中修改这一行：

```
<!-- 修改前 -->
<uni-grid :column="4" :highlight="false" :square="false" class="grid-container">

<!-- 修改后 -->
<uni-grid :column="4" :highlight="false" :square="false" :show-border="false" class="grid-container">
```

### 方案2：通过 CSS 强制覆盖

如果您不想修改模板，可以在样式表中添加：

```
.grid-container {
  padding: 20rpx 0;
  
  // 深度选择器覆盖 uni-grid 的内置边框
  :deep(.uni-grid) {
    border: none !important;
  }
}
```

## 推荐完整修改

建议同时使用方案1和方案2，确保边框被完全移除：

```
<!-- 模板部分 -->
<uni-grid 
  :column="4" 
  :highlight="false" 
  :square="false" 
  :show-border="false"  <!-- 添加这个属性 -->
  class="grid-container"
>
```

```
/* 样式部分 */
.grid-container {
  padding: 20rpx 0;
  
  // 双重保障
  :deep(.uni-grid) {
    border: none;
  }
}
```
