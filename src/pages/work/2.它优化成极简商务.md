### 🛠 替换你原来的 `<style>`部分为这个：

```
/* stylelint-disable selector-type-no-unknown */

  background: #f5f5f5;
}
/* stylelint-enable selector-type-no-unknown */

.work {
  padding: 20rpx 0;
}

.card-container {
  margin: 16rpx 20rpx;
  border: 1rpx solid #ddd; // 外层细边框
  background: #fff;
  border-radius: 0; // 无圆角
  box-shadow: none; // 无阴影
}

.grid-container {
  padding: 16rpx 0;
}

.grid-item {
  padding: 12rpx 0;
  border-right: 0.5rpx solid #eee; // 内部极细分隔线
  &:last-child {
    border-right: none;
  }
}

.nav-item {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  width: 100%;
  height: 100%;
  padding: 16rpx 8rpx;
  box-sizing: border-box;
  background: #fff;
  transition: background 0.1s ease;

  &:active {
    background: #f5f5f5; // 点击微灰度
  }
}

.icon-container {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 64rpx;
  height: 64rpx;
  margin-bottom: 12rpx;
}

.item-icon {
  width: 64rpx;
  height: 64rpx;
  border-radius: 0; // 无圆角
}

.item-text {

@media (max-width: 768px) {
  .grid-container {

  }

// 无点击缩放，只改背景

  transform: none;
}
```
