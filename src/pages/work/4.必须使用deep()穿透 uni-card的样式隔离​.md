# 间距问题分析与解决方案

根据您提供的代码和截图，我发现了问题所在：**`uni-card`组件本身有默认的外边距(margin)，而您的 `.card-container`类没有覆盖这个默认样式**。

## 🔍 问题原因

在您的代码中：

```
<uni-card :title="item.title" class="card-container">
```

虽然您定义了 `.card-container`类，但 `uni-card`组件内部可能有自己的默认样式，特别是：

- `uni-card`默认有 `margin-bottom: 20rpx`（或类似值）
- 您的 `.card-container`只设置了 `margin: 20rpx`，这会影响卡片本身，但**不会覆盖卡片之间的间距**

## ✅ 正确的解决方案

### 方法1：深度覆盖 `uni-card`的默认样式（推荐）

```
<style scoped>
.work {
  /* 确保父容器没有额外padding */
  padding: 0;
}

/* 深度选择器覆盖 uni-card 的默认样式 */
:deep(.uni-card) {
  margin: 10rpx 20rpx !important; /* 调整这个值来改变卡片间距 */
  border: none !important;
  box-shadow: 0 4rpx 12rpx rgba(0, 0, 0, 0.1) !important;
  border-radius: 16rpx !important;
}

/* 如果需要进一步减小间距 */
:deep(.uni-card:last-child) {
  margin-bottom: 10rpx !important;
}

.card-container {
  /* 这个类可以保留，但主要用于其他样式 */
  border-radius: 16rpx;
}

.grid-container {
  padding: 10rpx 0; /* 减小网格内部间距 */
}

.grid-item {
  padding: 10rpx 0; /* 减小网格项间距 */
}
</style>
```

### 方法2：使用更具体的选择器

```
.work :deep(.uni-card) {
  margin: 10rpx 20rpx !important;
}
```

### 方法3：如果上述方法无效，检查全局样式

在 `App.vue`中检查是否有全局设置：

```
/* App.vue */
.uni-card {
  margin: 10rpx 20rpx !important;
}
```

## 📌 关键修改点

1. **必须使用 `:deep()`穿透 `uni-card`的样式隔离** - 因为 `uni-card`是自定义组件，它的样式默认是作用域隔离的
2. **添加 `!important`** - 确保覆盖默认样式
3. **调整 `margin`值** - 从 `20rpx`改为 `10rpx`或更小
4. **检查父容器 `.work`的 padding** - 确保没有影响

## 🔧 为什么您之前的修改无效？

您可能只修改了 `.card-container`的样式，但 `uni-card`组件本身有默认的 `margin`，而 `.card-container`是放在 `uni-card`内部的类，**它不会影响卡片之间的间距**。

正确的方式是覆盖 `uni-card`本身的样式，而不是其内部类的样式。

## 💡 额外建议

如果您使用的是 uni-app 官方组件，可以参考其样式文档：

- `uni-card`默认有 `margin-bottom: 20rpx`
- 可以通过 `customStyle`属性传递样式，但不如深度选择器方便

试试这个方案，应该能解决间距问题！
